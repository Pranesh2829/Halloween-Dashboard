# Halloween-Dashboard
This dashboard analyzes participation trends in Halloween events to assist in effective event planning. It offers insights into weather conditions, predicts future event attendance, tracks year-over-year variations, and visualizes overall visitor turnout.

Description
This project showcases an interactive Halloween-themed analytics dashboard that examines historical participation patterns, audience behavior, and attendance forecasts. The dashboard leverages dynamic visualizations and data insights to enable informed planning and strategic decision-making.
